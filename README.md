# Garcia Intelligence — Site Institucional

Landing page da **Garcia Intelligence**, empresa de tecnologia com foco em desenvolvimento com IA, automações inteligentes e criação de SaaS.

---

## Sobre o projeto

Site institucional desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências externas. O objetivo foi criar uma página leve, tecnológica e objetiva, acessível tanto para clientes técnicos quanto para pessoas sem experiência em tecnologia.

---

## O que tem no site

- **Hero com animação em canvas** — três monitores com código rolando em tempo real e silhueta de desenvolvedor
- **Ticker animado** com os serviços e tecnologias da empresa
- **Seção de números** com diferenciais do negócio
- **Portfólio de serviços** com os 4 pilares da Garcia Intelligence
- **Processo em 4 passos** explicado de forma simples
- **CTA com terminal animado** e botões de contato direto
- **Logo com efeito typewriter** ao carregar a página
- **Totalmente responsivo** para mobile

---

## Tecnologias

- HTML5
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (Canvas API, IntersectionObserver)
- Google Fonts — Plus Jakarta Sans + JetBrains Mono
